WWW: "The World Wide Web"

The "web" is composed of several key components. The first is what you see as the user. You're using a computer. This computer has a "web browser" - the software you use to access the internet experience. If you're old enough, you might remember "Netscape" which was esseentially the first modern web browser. We now use Apple's Safari, Google's Chrome, and many others to access the internet.

The next component is the "internet" itself. The internet is composed of computers called servers. These servers are usually in a nondescript building, stacked in "racks", and maintained by some really smart computer IT technicians. These computers "host" the HTML web pages that we access every time we use browse the internet. The key take away is that the web is a collection of web pages

HTML: "Hypertext Markup Language"

  • Text - Basically, refers to the data that you see in your web browser
  • Markup - Arrangement of the content
  • Hyperlinks - "Links" your current page to other pages and information sources.

    • HTML is made up of "tags". These tags are used to form "elements". Elements are used to infuence the behavior, look, and feel of web pages.

    For example, a generic element might look liks this:

    <tag> contents </tag>

    The <tag> is the "opening tag"
    The </tag> is the "closing tag" So if you want to display the words "Run like the wind!" using bold letters, you can use the bold <b> tags to create an element like so

    <b>Run like the wind!</b>

    For the user, your HTML code will look like this: "Run like the wind!"

    Anchor Tag
    Another HTML tag is the "anchor tag" denoted by "a". The anchor tag allows you to create links to other pages or data sources on the web. The pseudocode for an anchor tag looks like this:

    <a href = attribute > contents </a>

    In this example, the "href" stands for "hypertext reference".

    So let's say you wanted to create a hyperlink that would take you to www.usatoday.com. It would look something like the following.

    <a href = www.usatoday.com > usatoday.com </a>

    Break Tag
    Not all elements use a closing tag. Tags that do not require a closing tag are called void tags. For example, the tag to create a line break or to push text to the next line on the page is called the "break" tag. The break tag is denoted by "br". So, all you would need to write in the HTML5 code is the following.

    < br >
    One last point about "br". The break tag is considered "inline" versus "block". The difference between the two will be discussed later in this lesson.
    Image Tag
    The image "img" tag, only uses the opening tag plus a source attribute. This source reference is denoted by "src". The source is usually the location of the image file for its associated element. The element also needs to designate a value for the source. This is noted below by "url". Hence, an image element might look something like this

    < img src = url > or < img src = alt >

    where the "alt" corresponds to some form of text. Remember, image elements do not have a closing tag.

    Let's say you wanted to put a line break into your webpage's text, then you would use the break tag < br >. This tag is also a void tag. Hence, using multiple < br > after a particular piece of text will lead to multiple line breaks.
    Paragraph Tag
    As discussed before, the "br" tag creates line breaks. It is considered an "inline" element. Inline elements are usually for modifying short lines of code. In contrast, the paragraph tag denoted by "p" creates an invisible boundary arond the code that you're working on. This kind of element is considered a "block" element. Block elements are used for groups of code or text.
    Span (Inline) vs Div (Block)
    These tags do not do anything specifically like say the bold tag. However, they allow a specific piece of information to be modified by CSS code. Similar to "br" tag, the "span" tag is inline. Hence, it's used for small chunks of code. Just like "paragraph" tag that created an invisible border around the code or text, you can use the "div" tag to do the same. Div is a block element used for large chunks of html code.
